Generalized Linear Model

广义线性模型

1.1 指数分布族（The Exponential Family）

对于因变量$y$，如果其概率分布满足一下指数分布，则该分布就称作指数分布：

$p_(y;\eta) = b(y)exp(\eta^TT(y) - \alpha(\eta))$

$\eta$: 分布的自然参数（natural parameter）或者称为标准参数
（canonical parameter）

$T(y)$: 充分统计量，通常用 $T(y) = y$

$\alpha(\eta))$: 对数分割函数（log partition function）

$exp(-\alpha(\eta))$:本质上是一个归一化常数，确保概率和为1

当$T(y)$被固定时，$a(\eta)$、$b(y)$就定义了一个以$\eta$为参数的一个指数分布。我们变化$\eta$就得到这个分布的不同分布。

$p_(y;\eta) = b(y)exp(\eta^Ty - \alpha(\eta))$ (1.1)

1.2 指数分布的退化形式

1.2.1 Bernoulli分布

伯努利分布即两点分布：
$p_(y;\phi)$
$= \phi^y(1-\phi)^(1-y)$
$= exp(log(\phi^y(1-\phi)^(1-y))$
$= exp(ylog\phi + (1-y)log(1-\phi))$
$= exp(y(log(\phi/(1-\phi)) + log(1-\phi)))$

对比(1.1)可有：
$b(y)=1$
$\eta=log(\phi/(1-\phi)$
$\phi=1/(1+exp(\eta))$
$\alpha(\eta)=-log(1-\phi)=log(1+exp(\eta))$
